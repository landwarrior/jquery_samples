CSSの確認

CSSの指定は、style属性を使って要素に直接書くのが最も強い。
その次がid指定、その次がclass指定、その次が要素指定。
例えば以下のようなスタイルをspan要素に当てる場合。 
/* id指定 */
#id-test {
  color: red;
}
/* class指定 */
.class-test {
  color: green;
}
/* 要素指定 */
span {
  color: blue;
}
対象の要素は以下のように記載。 
<span id="id-test" class="class-test">ほげほげふーばー$lt;/span>
ほげほげふーばー
idを取り除くと。
ほげほげふーばー
classも取り除くと。
ほげほげふーばー

※実際にはsample1というCSSのクラス配下のみにスタイルが当たるように記載している
